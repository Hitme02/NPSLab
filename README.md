# RVCE 6th Sem Network Programming Security Programs

<p>RV College of Engineering, Bengaluru</p>
<p><strong><span style="font-family: serif; font-size: 28px;">Network Programming Security (CS362IA)</span></strong></p>


<p><br></p>

## Lab Programs

### Program 1 — Implement a client and server communication using sockets programming.

### Program 2 — Write a program to implement Distance Vector routing protocol for a simple topology of routers.

### Program 3 — Write a program to implement error detection and correction concepts using Checksum and Hamming code.

### Program 4 — Implement a simple multicast routing mechanism.

### Program 5 — Write a program to implement a concurrent chat server that allows currently logged in users to communicate with one another.

### Program 6 — Implementation of concurrent and iterative echo server using both connection-oriented and connectionless socket system calls.

### Program 7 — Implementation of remote command execution using socket system calls.

### Program 8 — Write a program to encrypt and decrypt data using RSA and exchange the key securely using Diffie–Hellman key exchange protocol.
